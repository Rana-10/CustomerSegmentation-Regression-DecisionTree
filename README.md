## üîç Project overview
This project explores customer purchase behaviour in the electronics category to:
- Predict **average spending per purchase** (regression).
- Classify / understand features affecting purchase decisions (decision tree).
- Segment customers into groups using **K-Means** for targeted marketing.

Key outcomes:
- High-performing Linear Regression model for numeric prediction (report: strong accuracy).
- Decision Tree model yields ~**73%** accuracy for categorical tasks (report).
- K-Means clustering reveals actionable customer segments for inventory and marketing optimization.
